Beginner’s Guide to Python<\/h1>\n\n\n\n

New to programming? Python is free and easy to learn if you know where to start! This guide will help you to get started quickly.<\/p>\n\n\n\n

Chinese Translation<\/a><\/p>\n\n\n\n

New to Python?<\/h2>\n\n\n\n

Read BeginnersGuide\/Overview<\/a> for a short explanation of what Python is.<\/p>\n\n\n\n

Getting Python<\/h2>\n\n\n\n

Next, install the Python 3 interpreter on your computer. This is the program that reads Python programs and carries out their instructions; you need it before you can do any Python programming. Mac and Linux distributions may include an outdated version of Python (Python 2), but you should install an updated one (Python 3). See BeginnersGuide\/Download<\/a> for instructions to download the correct version of Python.<\/p>\n\n\n\n

There are also Python interpreter and IDE bundles available, such as Thonny<\/a>. Other options can be found at IntegratedDevelopmentEnvironments<\/a>.<\/p>\n\n\n\n

At some stage, you’ll want to edit and save your program code. Take a look at HowToEditPythonCode<\/a> for some advice and recommendations.<\/p>\n\n\n\n

Learning Python<\/h2>\n\n\n\n

Next, read a tutorial and try some simple experiments with your new Python interpreter.<\/p>\n\n\n\n

  • If you have never programmed before, see BeginnersGuide\/NonProgrammers<\/a> for a list of suitable tutorials.<\/li>
  • If you have previous programming experience, consult BeginnersGuide\/Programmers<\/a>, which lists more advanced tutorials.<\/li>
  • If English isn’t your first language, you might be more comfortable with a tutorial that’s been translated into your language. Consult python.org’s list of Non-English resources<\/a>.<\/li><\/ul>\n\n\n\n

    Most tutorials assume that you know how to run a program on your computer. If you are using Windows and need help with this, see How do I Run a Program Under Windows<\/a>.<\/p>\n\n\n\n

    Some sites offer in-browser coding for those who want to learn Python:<\/p>\n\n\n\n

    • Codecademy<\/a><\/li>
    • Coding Bootcamps<\/a><\/li>
    • DataCamp<\/a><\/li>
    • Dataquest<\/a> for Python for data science.<\/li>
    • HackInScience<\/a> free and open source platform.<\/li>
    • High School Technology Services<\/a> for general Python<\/li><\/ul>\n\n\n\n

      Print a cheat sheet<\/a> of the most important Python features and post it to your office wall until you know the basics well.<\/p>\n\n\n\n

      Once you have read a tutorial, you can browse through Python’s online documentation<\/a>. It includes a tutorial<\/a> that might come in handy, a Library Reference<\/a> that lists all of the modules that come standard with Python, and the Language Reference<\/a> for a complete (if rather dry) explanation of Python’s syntax.<\/p>\n\n\n\n

      When you are ready to write your first program, you will need a text editor<\/a> or an IDE<\/a>. If you don’t want to use Thonny or something more advanced, then you can use IDLE<\/a>, which is bundled with Python and supports extensions<\/a>.<\/p>\n\n\n\n

      This Python wiki also contains a page about Python One-Liners<\/a> — an obscure but interesting subculture in Python.<\/p>\n\n\n\n

      Need Help?<\/h2>\n\n\n\n

      Need help with any of this? Read BeginnersGuide\/Help<\/a> for mailing lists and newsgroups.<\/p>\n\n\n\n

      Most Python books will include an introduction to the language; see IntroductoryBooks<\/a> for suggested titles.<\/p>\n\n\n\n

      Consult BeginnersGuide\/Examples<\/a> for small programs and little snippets of code that can help you learn.<\/p>\n\n\n\n

      Or, if you prefer to learn Python through listening to a lecture, you can attend a training course or even hire a trainer to come to your company. Consult the PythonEvents<\/a> page to see if any training courses are scheduled in your area and the PythonTraining<\/a> page for a list of trainers.<\/p>\n\n\n\n

      Teachers can join the EDU-SIG<\/a>, a mailing list for discussion of Python’s use in teaching at any level ranging from K-12 up to university.<\/p>\n\n\n\n

            • The first place to look is the Python Package Index<\/a>.<\/li>
